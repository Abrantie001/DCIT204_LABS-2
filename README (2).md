## 1.FindMax.java
This program finds the maximum value in an array and identifies the index where it is located. 
It utilizes a linear search approach, iterating through the array once to track the highest value found.
## 2.PairSum.java
This program searches for a unique pair of numbers in an array that add up to a specific user-defined target sum. It implements a Brute Force (Nested Loop) algorithm with a time complexity of $O(n^2)$.

## How to Run the Programs
Ensure you have the Java Development Kit (JDK) installed on your machine.
